# README
## Project Title: Captain: 7 Seas
## Project Members: Noah Gibb, Kevin Fudge, Ruben Gonzalez, Kaylee Hernandez
## Platform: VR
## Hardware Requirements: Oculus Quest 2
## Target Audience: Anyone who enjoys pirates, sailing, sword or pistol combat, and VR
## Game Synopsis: You're the captain of a pirate ship, you're tasked with steering your ship to avoid obstacles and defending it against various types of enemies who are boarding to steal your cargo.
